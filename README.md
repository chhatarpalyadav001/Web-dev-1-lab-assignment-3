Responsive Portfolio Website
Description
This is a one-page responsive portfolio website built using Flexbox, CSS Grid, media queries, animations, and transitions.

Features
Flexbox for header & hero layout
CSS Grid for skills & projects
Fully responsive
Hover effects & smooth transitions
Keyframe animations
Breakpoints Tested
320px (Mobile)
768px (Tablet)
1024px (Laptop)
How to View
Open index.html in browser.
